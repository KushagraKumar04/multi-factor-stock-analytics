# Multi-Factor Stock Market Intelligence Platform

## Overview
This project is an enterprise-grade stock market analytics platform designed to analyze live financial data using advanced feature engineering, statistical analysis, machine learning, and explainable AI techniques.

## Key Features
- Live multi-source stock data ingestion
- Advanced technical & statistical indicators
- Market regime detection (Bull/Bear/Sideways)
- Risk & volatility analytics
- Explainable machine learning predictions
- Interactive Streamlit dashboard

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost, TensorFlow
- SHAP
- Streamlit & Plotly
